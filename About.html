LibrarySystem is a software developed to facilitate management of libraries of schools, colleges. Alike to any other library
management software it is empowered to perform the basic operations of library :- Inserting Books, Inserting Borrowers,
Handling transactions etc but it is its other features which makes the software robust. Some of the features imbibed in this
software are :


             • Ease of use.

             • Ability to change themes,fonts,background colours etc.

             • Ability to create database backups,load the same without bothering the user of any tedious backend handling.

             • Enhanced security features like secure login etc.

             • Lesser priveleges to the guest account.

             • Ability to run in most Operating systems viz Windows XP, Windows Vista, Windows 7, Linux, MacOS etc.


Lastly the greatest appraisal of our work would lie in the hands of our consumers if this software could be of any use to them.
